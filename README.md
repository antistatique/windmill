# Windmill

## Getting Started

### Install dependencies

```bash
yarn install
```

### Environment Variables

Create a `.env.local` file similar to `.env.local.example` and fill in the values for your environment.

```bash
cp .env.local.example .env.local
```

### Run the development server

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Test

```bash
yarn test
```
